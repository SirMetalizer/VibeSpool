# 📋 VibeSpool Backlog & Roadmap

Diese Liste enthält alle geplanten Features, Verbesserungen und bekannten Bugs, die durch das Feedback der Community gesammelt wurden. 

## 🎉 Kürzlich abgeschlossen (v1.9.5 & v1.9.6)
- [X] **Etiketten-Drucker / Label Creator:** Neues Tool zum Generieren von druckfertigen Etiketten (PNG) inkl. QR-Code, Farbe, Material und Drucktemperaturen.
- [X] **Doppeltiefe Regale:** Optionale Aktivierung von "Vorne (V)" und "Hinten (H)" Plätzen für tiefe Regalsysteme (inkl. Auto-Migration alter Bestände).
- [X] **Drag & Drop Quick-Swap:** Visuelles Tauschen von Spulen im Regal-Visualizer per Drag & Drop (inkl. Dropzone für das LAGER).
- [X] **Settings-Footer Fix:** Der "Änderungen Speichern"-Button im Einstellungsmenü bleibt nun immer sichtbar.
- [X] **Live-Sortierung & AMS-Prio:** Die Haupttabelle sortiert sich bei Umbuchungen nun live neu und das AMS klebt immer an oberster Stelle.
- [X] **Smart Color-Picker & Auto-Format:** Ein einzelner Farb-Button, der Hex-Codes mit einer riesigen internen Farbdatenbank abgleicht (Auto-Naming auch bei Tastatureingabe).
- [X] **Isolierte V2 Regal-Benennung:** Regalfächer können nun pro Regal individuell benannt werden (keine globalen Überschreibungen mehr).
- [X] **Gelöschte Regale retten:** Werden Regale in den Einstellungen verkleinert oder gelöscht, bucht VibeSpool darin liegende Spulen automatisch ins "LAGER" um.
