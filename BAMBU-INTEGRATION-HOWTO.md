📖 1. Anleitung: Bambu Lab Drucker in VibeSpool integrieren

**Voraussetzung:**

VibeSpool kommuniziert lokal über dein Heimnetzwerk (WLAN/LAN) mit deinem Drucker. Der Drucker muss eingeschaltet und im selben Netzwerk wie dein PC sein.

**So findest du deine Zugangsdaten:**

Gehe direkt an das Display deines Bambu Lab Druckers (X1, P1, A1 Serie) oder öffne die Bambu Handy App / Bambu Studio.
Drucker IP-Adresse: * Am Drucker: Gehe auf das Zahnrad (⚙️ Einstellungen) ➔ Netzwerk. Dort steht die aktuelle IP (z.B. 192.168.178.XX).

**Access Code (Zugangscode):**

Am Drucker: Gehe auf ⚙️ Einstellungen ➔ Netzwerk. Dort findest du den Access Code (oft ein 8-stelliger Code).
Hinweis: Der "Nur LAN-Modus" muss nicht zwingend aktiviert sein. Der Code reicht aus!

**Seriennummer:**

Am Drucker: Gehe auf ⚙️ Einstellungen ➔ Allgemein ➔ Geräte-Info. Die Seriennummer beginnt meist mit 00M..., 01S... oder ähnlich.
Alternativ: Im Bambu Studio unter dem Reiter "Gerät" ➔ "Update" oder in der Handy-App.

Trage diese drei Werte in VibeSpool unter ⚙ Optionen ➔ 🤖 Drucker ein, setze den Haken bei "Bambu AMS Live-Sync aktivieren" und starte VibeSpool einmal neu!
